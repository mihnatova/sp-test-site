Data Collection Test Form

This form collects personal data fields typically found on websites. Use it to test how Secure Privacy handles data collection consent.

Note: This is a static test form. No data is actually transmitted. The form is for testing consent UX flows — does the user see proper consent mechanisms before submitting personal data?

What to Test

  1. Does the consent banner appear before the user can interact with the form?
  2. Is the newsletter checkbox separate from cookie consent?
  3. Does the Privacy Policy link work correctly?
  4. Test with different countries selected — does geo-targeting change the banner?
  5. Can the form be submitted if cookie consent is declined?
